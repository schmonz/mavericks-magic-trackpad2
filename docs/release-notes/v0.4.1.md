## Mavericks Trackpad 2 — v0.4.1

A quick fix for the in-pane update button.

- **"Check for Updates" now opens the updater reliably** — in 0.4.0 the button could appear to do
  nothing: the updater started but its window never came to the front, so no dialog showed. It now
  consistently brings up the update dialog.

Requires macOS 10.9.5 or later (the 10.9 series).
