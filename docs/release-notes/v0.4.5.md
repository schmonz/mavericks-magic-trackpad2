## Mavericks Trackpad 2 — v0.4.5

Fixes so updates actually update everything — and a quieter update checker.

- **Updates now replace the driver itself.** Because of an installer version-check quirk, earlier
  updates could refresh the helper apps but silently skip the kernel extension — the actual trackpad
  driver — leaving it at the previously-installed version. Updates now always install every component.

- **The background update check is silent and never lingers.** The scheduled daily check no longer
  pops a window or gets stuck in the Dock. It quietly notes whether an update is available (shown in
  **Trackpad ▸ About**); you install on your own terms with "Check for Updates."

Note: this is the release that unsticks the driver update — installing 0.4.5 brings the driver current
even if a previous version had left it behind. (On install you may see the standard 10.9 notice that
the kernel extension "is not from an identified developer but will still be loaded" — expected for this
unsigned driver; it loads normally.)

Requires macOS 10.9.5 or later (the 10.9 series).
