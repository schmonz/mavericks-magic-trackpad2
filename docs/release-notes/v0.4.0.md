## Mavericks Trackpad 2 — v0.4.0

A dedicated About tab, and a polished update experience.

- **New "About" tab in the Trackpad preference pane** — one place for everything about the driver:
  check for updates, turn automatic checking on or off, see which version you're running, and open the
  project on GitHub. When a newer version is available, the tab tells you.
- **Clearer updates** — checking when you're already current now shows a proper "You're up to date"
  message (instead of doing nothing visible); after an update installs, a confirmation tells you which
  version you're now on and the trackpad is ready to use with no tap needed; release notes are properly
  formatted (and also appear on the GitHub release page); and the updater shows the product name and a
  trackpad icon rather than a generic helper.
- **Optional automatic update checks** — turn on "Check automatically" in the About tab and the driver
  quietly checks for new versions on its own; leave it off and nothing checks. Off by default.
- **Connect/disconnect shows a trackpad** — the on-screen badge shown when the Magic Trackpad 2
  connects or disconnects over Bluetooth now pictures a trackpad, not a mouse.
- **The installer explains itself** — the installer now states up front why it asks for your
  administrator password (it installs a system driver, used only to drive your trackpad).

Requires macOS 10.9.5 or later (the 10.9 series).
