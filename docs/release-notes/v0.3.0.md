## Mavericks Trackpad 2 — v0.3.0

One-click updates, a real device name you can set yourself, and steadier connections.

- **Check for Updates, in the pane** — the Trackpad preference pane now has a one-click "Check for
  Updates" control that fetches and installs signed releases (no Terminal, no reboot).
- **Rename your trackpad — and it sticks to the device** — renaming the Magic Trackpad 2 in the
  Bluetooth pane now writes the name onto the device itself, so it persists across reboots and even
  follows the trackpad to another Mac. (Previously the name was only a local nickname on this Mac.)
- **Smoother Bluetooth reconnect** — after a Bluetooth power-cycle the trackpad re-enables reliably
  instead of occasionally getting stuck without a cursor.
- **No-click USB → Bluetooth handoff** — unplug the cable and the trackpad resumes over Bluetooth
  immediately, with no tap needed to wake it.
- **Steadier USB plug-in** — hardened the USB bring-up path that could, in rare cases, destabilize the
  system when switching from Bluetooth to cable.

Requires macOS 10.9.5 or later (the 10.9 series).
