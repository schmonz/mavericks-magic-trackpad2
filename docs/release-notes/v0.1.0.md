## Mavericks Trackpad 2 — v0.1.0

First preview release: a Magic Trackpad 2 driver for macOS 10.9 Mavericks.

- **Full gestures over USB and Bluetooth** — cursor, scroll, two- and three-finger swipes, tap,
  physical click, two-finger secondary click, tap-to-click.
- **Correct pad geometry** — genuine sensor grid, so no edge dead zones.
- **System Preferences integration** — the Trackpad pane tracks live USB/Bluetooth switches, shows
  the real battery percentage on both transports, and displays the trackpad's own icon.
- **Self-update** — this build checks for and installs updates in place.

Requires macOS 10.9.5 or later (the 10.9 series).
