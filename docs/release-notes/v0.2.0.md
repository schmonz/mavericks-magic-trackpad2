## Mavericks Trackpad 2 — v0.2.0

A more polished System Preferences experience, plus the groundwork for in-place updates.

- **Trackpad pane** — shows the real battery percentage on both USB and Bluetooth, tracks live
  USB↔Bluetooth switches without stale or flickering state, recovers the battery reading after a
  Bluetooth power-cycle, and hides the never-applicable "Change Batteries" prompt.
- **Bluetooth pane** — the Magic Trackpad 2 shows the trackpad's own icon instead of a generic
  Bluetooth logo.
- **Self-update groundwork** — the updater is now bundled and the signed release feed is in place. A
  one-click "Check for Updates" control in the pane arrives in a later release.

Requires macOS 10.9.5 or later (the 10.9 series).
