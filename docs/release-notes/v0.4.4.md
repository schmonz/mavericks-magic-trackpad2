## Mavericks Trackpad 2 — v0.4.4

A background-updater reliability fix, plus internal cleanup — and the first release you can
update to in-pane from 0.4.3.

- **The daily update check no longer gets stuck.** A scheduled background check that stalled could
  leave the updater sitting in the Dock with no window, refusing to quit. It now always exits: a
  watchdog force-quits a check that never resolves, and every step is logged so a stuck run can be
  diagnosed. Manual "Check for Updates" from the pane is unaffected.

- **Under the hood.** Consolidated the Bluetooth ↔ USB transport tracking — the pane's view
  switching *and* the unplug-wakes-Bluetooth handoff — onto one shared component, on top of earlier
  engine and pipeline unification. No change to how the trackpad behaves: same gestures, clicks, and
  transport switching.

- **The driver now reports its real version** (it previously always reported 1.0.0).

From 0.4.3, use **Trackpad ▸ About ▸ Check for Updates**.

Requires macOS 10.9.5 or later (the 10.9 series).
