## Mavericks Trackpad 2 v0.5.1

- Fixes a leftover from the previous version's internal rename: updating
  from an older release could leave a second, stale "About" pane behind
  in the Trackpad preference pane. This release removes the old companion
  loader during the update, so only the current pane remains.

- Preserves your "Automatically check for updates" choice across the
  update, so it no longer resets when you upgrade.
