## Mavericks Trackpad 2 v0.5.1

- The driver is now built on a 10.9 backport of VoodooInput, the
  open-source multitouch framework. This makes it easy to add other
  multitouch drivers besides the Magic Trackpad 2, and opens a path
  to potentially supporting even earlier versions of Mac OS X and/or
  contributing these backports back to upstream VoodooInput.

- The Trackpad preference pane's About tab will show when you need
  to restart for an update to take effect.
