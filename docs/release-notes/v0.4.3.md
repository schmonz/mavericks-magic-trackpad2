## Mavericks Trackpad 2 — v0.4.3

Makes the in-pane updater actually install updates.

- **Self-update now installs reliably** — earlier versions could check for and download an update but
  then hang without installing it. The update helper is now a standard app launched the normal way, so
  the install runs to completion (with the usual admin-password prompt, since it installs a driver).

Note: because older versions can't reliably update *themselves* to this one, install 0.4.3 once by hand
(download the .pkg below). After that, in-pane updates work normally.

Requires macOS 10.9.5 or later (the 10.9 series).
